## Common Tasks

### Switch to a Different Match
1. Tap **?** (hamburger menu)
2. Tap the **match** you want to work on
3. Menu closes, you're now editing that match

### Add a New Match
1. Tap **?**
2. Tap **+ Add** (top right of matches section)
3. New match created and selected

### Duplicate a Match
1. Tap **?**
2. Find the match you want to copy
3. Tap the **?? duplicate icon**
4. New copy is created

### Delete a Match
1. Tap **?**
2. Find the match to delete
3. Tap the **?? delete icon**
4. Confirm deletion

### Configure Actions
1. Tap **?**
2. Tap **Configuration**
3. Tap **Configure Actions**
4. Configuration modal opens

### Export All Matches
1. Tap **?**
2. Tap **Export All Matches**
3. JSON file downloads

### Save as Template
1. Tap **?**
2. Tap **Save as Template**
3. Enter template name
4. Tap **Save**

### Load a Template
1. Tap **?**
2. Tap **Load Template** (shows count)
3. Browse templates
4. Tap **Load** on desired template
5. Confirm if prompted

## Visual Indicators

### Current Match
- **Blue ring** around the match card
- **Indigo background** color

### Alliance Colors
- **Red badge** for red alliance
- **Blue badge** for blue alliance

### Match Info
- **Match number** (e.g., "Match #1")
- **Partner team** (or "No partner")
- **Action count** (e.g., "5 actions")

### Template Count
- **Badge** on "Load Template" shows how many templates saved

## Navigation Tips

### Back Button
- Appears in **Configuration** and **Templates** submenus
- **? arrow** in top-left
- Tap to return to main menu

### Close Menu
- Tap **×** in any menu view
- Or tap the **dark overlay** outside menu
- Menu automatically closes after most actions

## Keyboard Shortcuts (Desktop)
- **Escape** - Close menu

## Troubleshooting

### Can't find a match
- Scroll down in the matches section
- All matches are listed at the top

### Menu won't close
- Tap the **×** button
- Or tap outside the menu

### Action didn't work
- Menu may have closed - reopen it
- Check if the action completed (e.g., match was added)

## Pro Tips

1. **Quick switching** - Hamburger menu makes it easy to jump between matches during a tournament

2. **One-tap duplicate** - Use the duplicate button when you have similar strategies

3. **Template workflow** - Save common strategies as templates, load before matches

4. **Export regularly** - Use "Export All Matches" to backup your configurations

5. **Current match highlighted** - Always know which match you're working on by the blue ring
